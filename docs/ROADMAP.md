# 🗺️ Roadmap

Future development plans organized by version.

## v1.0 — Current Release ✅

The initial release with core functionality:

- [x] Spotify OAuth login (PKCE)
- [x] Home screen (recently played, new releases, featured, top tracks)
- [x] Search (songs, artists, albums with debounce)
- [x] Mini player & full-screen player
- [x] Like songs (Supabase)
- [x] Playlist creation & management
- [x] User profile
- [x] Dark modern UI
- [x] Smooth animations
- [x] Complete documentation (15 docs)

## v1.1 — Enhanced Experience (Planned)

- [ ] Artist detail screen
- [ ] Album detail screen with track listing
- [ ] Playlist detail screen with reorderable tracks
- [ ] Queue view in full player
- [ ] Song lyrics integration
- [ ] Animated skeleton loading screens
- [ ] Share song/playlist via deep links
- [ ] Pull-to-refresh on all screens

## v1.2 — Social Features (Planned)

- [ ] Follow artists
- [ ] Public playlist sharing
- [ ] Activity feed
- [ ] Collaborative playlists
- [ ] User-to-user song sharing

## v2.0 — Advanced Features (Future)

- [ ] Background audio playback (react-native-track-player)
- [ ] Lock screen controls
- [ ] Download for offline playback
- [ ] Equalizer settings
- [ ] Sleep timer
- [ ] Crossfade between tracks
- [ ] Android Auto / CarPlay support
- [ ] Widget for home screen
- [ ] Push notifications for new releases

## v3.0 — Platform Expansion (Future)

- [ ] Apple Watch companion app
